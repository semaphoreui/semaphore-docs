## Who Is This For?

- **DevOps Engineers** looking for a lightweight automation platform
- **System Administrators** managing infrastructure with Ansible or Terraform
- **IT Teams** wanting to provide self-service automation to developers
- **Anyone** seeking an easy-to-use alternative to complex CI/CD platforms

## What Makes Semaphore Different?

Semaphore UI stands out by being:

- **Lightweight**: Single binary, minimal resources, no complex dependencies
- **Versatile**: Run Ansible, Terraform, scripts - all from one interface
- **Secure**: Self-hosted, encrypted credentials, role-based access control
- **Simple**: Install in minutes, intuitive UI, straightforward configuration
- **Powerful**: Remote runners, scheduling, REST API, Git integration

## Prerequisites

Before you begin, ensure you have:

- A Linux server, macOS machine, or Windows with WSL
- Basic familiarity with one or more of: Ansible, Terraform, Bash, or PowerShell
- (Optional) A Git repository with your automation code
