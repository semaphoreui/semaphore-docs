# User Guide

Learn how to use Semaphore day-to-day: create projects, run tasks, manage inventories, keys, variables, and more.

## Start here

1. [Organize work](./projects.md) – create projects, manage access, and review activity.
2. [Connect resources](./repositories.md) – add code, inventories, variables, and secrets.
3. [Build automation](./task-templates/README.md) – create task templates for Ansible, Terraform/OpenTofu, scripts, and more.
4. [Run & monitor](./tasks.md) – execute tasks, schedule repeats, and integrate notifications.
